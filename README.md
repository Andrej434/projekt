# 📝 To-Do List Web App (PHP + MySQL)

## 📌 Popis projektu

Táto aplikácia je jednoduchý To-Do List vytvorený v jazyku PHP s využitím databázy MySQL (mysqli).
Umožňuje používateľom registráciu, prihlásenie a správu vlastných poznámok (úloh).

---

## ⚙️ Použité technológie

* PHP (bez frameworku)
* MySQL (mysqli)
* HTML + Bootstrap 5

---

## 🗄️ Databáza

Projekt obsahuje SQL skript (`database.sql`), ktorý:

* vytvorí databázu
* vytvorí tabuľky:

  * `pouzivatelia`
  * `poznamky`
* vloží testovacie (dummy) dáta

---

## 🔧 Funkcionalita

### 👤 Používatelia

* registrácia
* prihlásenie
* odhlásenie

### 📝 To-Do list

* CREATE – uloženie poznámky
* READ – načítanie poznámky z databázy
* UPDATE – prepísanie poznámky
